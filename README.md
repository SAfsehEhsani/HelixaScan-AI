## 📋 Overview

HelixaScan-AI is an intelligent web application designed to assist doctors, dermatologists, and individuals in analyzing skin conditions. Using advanced AI algorithms, the platform provides accurate skin problem detection and treatment recommendations, making dermatological care more accessible and efficient.

**Live Application:** [View HelixaScan-AI](https://ai.studio/apps/7ccff246-f463-40f0-a690-467fa0ed2287)

## ✨ Features

- 🔬 **AI-Powered Analysis** - Advanced machine learning algorithms for accurate skin condition detection
- 👨‍⚕️ **Medical Professional Tools** - Specialized features for healthcare practitioners
- 📱 **User-Friendly Interface** - Intuitive design for both medical and non-medical users
- 🎯 **Treatment Recommendations** - Evidence-based suggested treatments and care protocols
- 🔒 **Secure & Privacy-Focused** - HIPAA-compliant infrastructure for sensitive medical data

## 🚀 Quick Start

### Prerequisites

- **Node.js** 16.x or higher
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SAfsehEhsani/HelixaScan-AI.git
cd HelixaScan-AI
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`
